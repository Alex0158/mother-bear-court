## 📐 技術架構摘要

### 技術棧

- **框架**：React 18.2+ + TypeScript 5.2+
- **構建工具**：Vite 5.0+
- **UI組件庫**：Ant Design 5.12+
- **狀態管理**：Zustand 4.4+ + React Query
- **路由**：React Router 6.20+
- **HTTP客戶端**：Axios 1.6+

### 項目結構

```
frontend/
├── src/
│   ├── components/     # 組件
│   ├── pages/          # 頁面
│   ├── hooks/          # 自定義Hooks
│   ├── store/          # 狀態管理
│   ├── services/       # API服務
│   ├── utils/          # 工具函數
│   ├── types/          # 類型定義
│   └── router/         # 路由配置
```
