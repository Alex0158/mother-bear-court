#### 4.4 交互設計

**表單驗證**：
- **實時驗證**：
  - 郵箱格式驗證
  - 密碼長度驗證（至少8位）
- **提交驗證**：
  - 檢查所有必填項
  - 顯示錯誤提示
- **錯誤提示**：
  - 輸入框下方顯示
  - 紅色文字 + 錯誤圖標
  - 輸入正確後自動消失

**登錄流程**：
1. 用戶填寫表單
2. 點擊「登錄」按鈕
3. 顯示加載動畫
4. 發送登錄請求
5. **成功**：
   - 保存Token到localStorage（如果選擇記住我）
   - 跳轉到目標頁面或首頁
   - 顯示成功提示
6. **失敗**：
   - 顯示錯誤提示
   - 保持表單內容不變

**響應式適配**：
- **桌面端**：居中卡片，固定寬度
- **移動端**：全屏寬度，調整間距和字號

#### 5.3 詳細設計說明

**進度指示器設計**：

- **樣式**：步驟條（Step）
- **步驟**：
  1. 填寫郵箱（當前步驟高亮）
  2. 驗證碼驗證
  3. 設置密碼
  4. 完成註冊
- **視覺**：
  - 已完成：溫暖橘背景，白色對勾
  - 當前：溫暖橘背景，白色數字
  - 未完成：淺灰背景，灰色數字

**步驟1：郵箱輸入**：

**表單內容**：
- **郵箱輸入框**：同登錄頁面
- **暱稱輸入框**（可選）：
  - 標籤：「暱稱（可選）」
  - 輸入框：標準輸入框
  - 占位符：「請輸入暱稱（2-20字符）」
- **下一步按鈕**：
  - 尺寸：100%寬度，高度56px
  - 樣式：主要按鈕
  - 文字：「發送驗證碼」

**交互**：
- 驗證郵箱格式
- 點擊「發送驗證碼」後，發送驗證郵件
- 顯示成功提示：「驗證碼已發送到您的郵箱，請查收」
- 自動進入步驟2

**步驟2：驗證碼驗證**：

**表單內容**：
- **驗證碼輸入框**：
  - 標籤：「驗證碼」
  - 輸入框：6位數字輸入框（特殊樣式）
  - 占位符：「請輸入6位驗證碼」
  - 樣式：大號字體（24px），居中，每個數字一個輸入框
- **倒計時提示**：
  - 位置：驗證碼輸入框下方
  - 文字：「驗證碼有效期：5分鐘，還剩 X 秒」
  - 字號：12px
  - 顏色：文字輔助色
- **重新發送按鈕**：
  - 位置：倒計時提示下方
  - 樣式：文字鏈接
  - 文字：「沒有收到？重新發送」
  - **禁用狀態**：倒計時未結束時禁用
- **下一步按鈕**：
  - 樣式：主要按鈕
  - 文字：「驗證並繼續」

**交互**：
- 自動聚焦第一個輸入框
- 輸入時自動跳轉到下一個輸入框
- 驗證碼格式驗證（6位數字）
- 倒計時結束後，可重新發送
- 驗證成功後，自動進入步驟3

**步驟3：設置密碼**：

**表單內容**：
- **密碼輸入框**：
  - 標籤：「設置密碼」
  - 輸入框：password類型
  - 占位符：「請輸入密碼（至少8位，包含字母和數字）」
  - 右側：顯示/隱藏密碼圖標
- **密碼強度指示器**：
  - 位置：密碼輸入框下方
  - 樣式：進度條（4個等級）
  - 等級：
    - 弱：紅色，< 8字符
    - 中：黃色，8-12字符，包含字母和數字
    - 強：綠色，> 12字符，包含字母、數字、特殊字符
    - 很強：深綠色，> 16字符，包含所有類型
- **確認密碼輸入框**：
  - 標籤：「確認密碼」
  - 輸入框：password類型
  - 占位符：「請再次輸入密碼」
  - 驗證：與密碼輸入框一致
- **完成註冊按鈕**：
  - 樣式：主要按鈕
  - 文字：「完成註冊」

**交互**：
- 實時顯示密碼強度
- 驗證兩次密碼輸入是否一致
- 密碼強度不足時，顯示提示
- 註冊成功後，自動進入步驟4

**步驟4：完成註冊**：

**內容**：
- **成功圖標**：對勾圖標（64x64px，成功綠）
- **標題**：「註冊成功！」
- **副標題**：「歡迎加入熊媽媽法庭」
- **提示文字**：「現在你可以使用完整功能了」
- **操作按鈕**：
  - 「開始配對」：主要按鈕，跳轉到配對頁面
  - 「稍後再說」：次要按鈕，跳轉到首頁

**交互**：
- 顯示成功動畫（對勾動畫）
- 3秒後自動跳轉到配對頁面（可取消）

#### 7.4 交互設計

**篩選交互**：
- 點擊篩選標籤，實時過濾方案
- 多個篩選條件可同時生效
- 顯示當前篩選結果數量

**方案選擇**：
- 點擊「選擇方案」按鈕
- 顯示確認彈窗：「確定選擇這個方案嗎？」
- 確認後，跳轉到方案詳情頁，可自定義調整

**方案預覽**：
- 點擊「查看詳情」按鈕
- 顯示方案詳情彈窗或側邊欄
- 可在此處直接選擇方案

**響應式適配**：
- **桌面端**：2列網格，完整功能
- **移動端**：1列網格，簡化部分功能
