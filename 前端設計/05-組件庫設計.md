## 🧩 組件分類

### 組件層級

1. **基礎組件**：按鈕、輸入框、標籤等基礎UI元素

## 📦 基礎組件設計

### 1. Button 按鈕組件

#### 1.1 組件概述

**用途**：觸發操作的主要交互元素

**變體**：
- Primary（主要按鈕）
- Secondary（次要按鈕）
- Text（文字按鈕）
- Link（鏈接按鈕）

---

#### 1.2 設計規范

**Primary按鈕**：
- **尺寸**：
  - Large：240x56px（桌面端），100%寬度，高度56px（移動端）
  - Medium：160x48px（桌面端），100%寬度，高度48px（移動端）
  - Small：120x40px（桌面端），100%寬度，高度40px（移動端）
- **樣式**：
  - 背景色：溫暖橘 (#FF8C42)
  - 文字顏色：白色
  - 字號：18px（Large），16px（Medium），14px（Small）
  - 字重：Medium
  - 圓角：8px
  - 陰影：中陰影
- **狀態**：
  - **默認**：正常樣式
  - **懸停**：背景色加深（#E67A2E），陰影增大
  - **點擊**：縮放0.98，觸覺反饋（移動端）
  - **禁用**：背景色淺灰 (#F5F5F5)，文字顏色文字輔助色，不可點擊
  - **加載**：顯示加載動畫（旋轉圖標），禁用點擊

**Secondary按鈕**：
- **尺寸**：同Primary按鈕
- **樣式**：
  - 背景色：透明
  - 邊框：2px solid 溫暖橘
  - 文字顏色：溫暖橘
  - 其他：同Primary按鈕
- **狀態**：
  - **懸停**：背景色填充（溫暖橘，10%透明度）
  - 其他：同Primary按鈕

**Text按鈕**：
- **尺寸**：自適應寬度，高度40px
- **樣式**：
  - 背景色：透明
  - 邊框：無
  - 文字顏色：溫暖橘
  - 字號：16px
- **狀態**：
  - **懸停**：背景色填充（溫暖橘，10%透明度），下劃線
  - 其他：同Primary按鈕

**Link按鈕**：
- **尺寸**：自適應寬度，高度自適應
- **樣式**：
  - 背景色：透明
  - 邊框：無
  - 文字顏色：溫暖橘
  - 字號：14px
  - 下劃線：默認無，懸停時顯示
- **狀態**：
  - **懸停**：文字顏色加深，下劃線
  - 其他：同Primary按鈕

---

#### 1.3 使用示例

**代碼結構**：
```typescript
<Button 
  type="primary" 
  size="large" 
  loading={false}
  disabled={false}
  onClick={handleClick}
>
  提交案件
</Button>
```

**圖標按鈕**：
```typescript
<Button 
  type="primary" 
  icon={<PlusIcon />}
  iconPosition="left"
>
  創建新案件
</Button>
```

### 2. Input 輸入框組件

#### 2.1 組件概述

**用途**：文本輸入的主要交互元素

**變體**：
- Text（文本輸入）
- TextArea（多行文本）
- Password（密碼輸入）
- Number（數字輸入）
- Search（搜索輸入）

---

#### 2.2 設計規范

**Text輸入框**：
- **尺寸**：
  - Large：100%寬度，高度56px
  - Medium：100%寬度，高度48px
  - Small：100%寬度，高度40px
- **樣式**：
  - 背景色：白色
  - 邊框：1px solid 邊框色 (#D9D9D9)
  - 圓角：8px
  - 內邊距：16px（左右），12px（上下）
  - 字號：16px
  - 行高：1.5
- **狀態**：
  - **默認**：正常樣式
  - **聚焦**：邊框顏色溫暖橘，邊框寬度2px，小陰影（溫暖橘色調）
  - **錯誤**：邊框顏色錯誤紅，右側顯示錯誤圖標
  - **禁用**：背景色淺灰，文字顏色文字輔助色，不可編輯
- **占位符**：
  - 顏色：文字輔助色 (#8C8C8C)
  - 字號：16px

**TextArea輸入框**：
- **尺寸**：
  - 最小高度：120px
  - 可拖拽調整高度（桌面端）
  - 固定高度，可滾動（移動端）
- **樣式**：同Text輸入框
- **特殊功能**：
  - 字數統計：右下角顯示「已輸入 X / 最大字數 字」
  - 自動保存：每30秒自動保存到localStorage

**Password輸入框**：
- **樣式**：同Text輸入框
- **特殊功能**：
  - 右側圖標：顯示/隱藏密碼切換圖標
  - 密碼強度指示器：下方顯示密碼強度（弱/中/強/很強）

**Search輸入框**：
- **樣式**：同Text輸入框
- **特殊功能**：
  - 左側圖標：搜索圖標
  - 右側圖標：清除按鈕（有內容時顯示）
  - 實時搜索：輸入時實時過濾結果

---

#### 2.3 使用示例

**代碼結構**：
```typescript
<Input
  type="text"
  size="large"
  placeholder="請輸入郵箱地址"
  value={email}
  onChange={handleChange}
  error={errorMessage}
  disabled={false}
/>
```

**帶標籤的輸入框**：
```typescript
<FormItem label="郵箱地址" required>
  <Input
    type="email"
    placeholder="請輸入郵箱地址"
    value={email}
    onChange={handleChange}
  />
</FormItem>
```

#### 6.1 組件概述

**用途**：展示母熊法官形象，增強品牌識別和情感連接

**變體**：
- Large（大尺寸，首頁Hero區域）
- Medium（中等尺寸，頁面標題區域）
- Small（小尺寸，卡片或表單中）

---

#### 6.2 設計規范

**Large尺寸**：
- **尺寸**：400x400px（桌面端），300x300px（移動端）
- **動畫**：懷抱姿勢，輕微的呼吸動畫（循環）
- **表情**：溫柔但堅定
- **顏色**：棕色主體 + 金色法袍點綴

- **尺寸**：200x200px（桌面端），150x150px（移動端）
- **動畫**：懷抱姿勢，輕微動作
- **表情**：根據上下文調整（溫和/理解/堅定）

**Small尺寸**：
- **尺寸**：120x120px（桌面端），80x80px（移動端）
- **動畫**：靜態或輕微動作
- **表情**：簡化版

**表情變體**：
- **溫和**：用於引導和說明
- **理解**：用於判決結果
- **堅定**：用於重要提示
- **幽默**：用於輕鬆場景

---

#### 6.3 使用示例

### 15. Toast 提示組件

#### 11.1 組件概述

**用途**：輕量級提示，用於操作反饋

**類型**：
- Success（成功）
- Error（錯誤）
- Warning（警告）
- Info（信息）

---

#### 11.2 設計規范

**整體樣式**：
- **尺寸**：自適應寬度，最小320px，最大500px
- **背景**：白色
- **邊框**：左側4px彩色邊框（根據類型）
- **圓角**：8px
- **內邊距**：16px
- **陰影**：中陰影

**內容布局**：
- **左側**：圖標（24x24px）
- **中間**：文字內容
- **右側**：關閉按鈕（可選）

**顏色**：
- **Success**：成功綠邊框，成功綠圖標
- **Error**：錯誤紅邊框，錯誤紅圖標
- **Warning**：警告黃邊框，警告黃圖標
- **Info**：信息藍邊框，信息藍圖標

**動畫**：
- **出現**：從右側滑入（0.3秒）
- **消失**：向右側滑出（0.3秒）

**位置**：
- 右上角（默認）
- 可配置：top-left, top-center, top-right, bottom-left, bottom-center, bottom-right

**持續時間**：
- 默認：3秒
- 可配置：1-10秒
- 手動關閉：點擊關閉按鈕

---

#### 11.3 使用示例

**代碼結構**：
```typescript
Toast.success("操作成功！", {
  duration: 3000,
  position: "top-right"
});
```
